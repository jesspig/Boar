# Boar - Compiler

这是一个自制的编译器，它用于编译 `Boar` 语言，它参考自 `TypeScript` ，所以语法看上去可能会有点相似，但是这只是暂定的，主要看开发难度，或许之后会变成其他的样子。所以可能会出现版本间完全不兼容的情况

- [ ] 基本运算 `[+-*/]`
- [ ] 函数调用 `func xxx...`
- [ ] 分支语句 `if...else...`
- [ ] 循环语句 `while...`
- [ ] 递归调用 `func xxx{...xxx...}`
